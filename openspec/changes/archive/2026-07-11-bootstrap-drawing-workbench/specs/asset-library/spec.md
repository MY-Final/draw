## ADDED Requirements

### Requirement: 素材库存储
系统 SHALL 将每张图片以 Blob 形式存入本地素材库,图字节与生成元数据分离,并以稳定 id 相互引用;素材默认不过期。

#### Scenario: 图片以 Blob 落库
- **WHEN** 一次生成产出图片
- **THEN** 每张图片作为独立 Blob 存储并分配唯一 id,生成记录以该 id 引用,不内联复制图字节

#### Scenario: 一图多用不复制
- **WHEN** 同一张图既是某次生成的输出、又被另一次生成用作参考图
- **THEN** 系统仅存储一份图字节,由多条记录以 id 引用

### Requirement: 素材预览与复用
系统 SHALL 展示素材库中的图片供预览,并允许将任一素材选作后续生成的参考图。

#### Scenario: 预览素材
- **WHEN** 用户打开素材库
- **THEN** 系统展示已存图片的缩略预览

#### Scenario: 选作参考图
- **WHEN** 用户在 chat 协议下从素材库选择一张图作为参考图
- **THEN** 该图被加入当前生成的参考图集合
